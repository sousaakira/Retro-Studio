Vamos analisar o que já temos e o que poderia ser útil adicionar. Atualmente as ferramentas implementadas são:

**✅ Ferramentas Implementadas:**
1. `read_file` - Lê conteúdo de arquivos
2. `list_directory` - Lista arquivos de um diretório
3. `search_files` - Busca arquivos por padrão glob
4. `grep_code` - Busca texto no código
5. `get_project_structure` - Estrutura completa do projeto
6. `get_file_info` - Informações sobre arquivo (tamanho, tipo, linhas)

**🔧 Ferramentas Úteis para Adicionar:**

### 1. **Ferramentas de Modificação** (Essenciais)
- `write_file` - Criar ou modificar arquivos
- `append_to_file` - Adicionar conteúdo ao final de arquivo
- `replace_in_file` - Buscar e substituir em arquivo
- `delete_file` - Deletar arquivo
- `create_directory` - Criar diretório
- `rename_file` - Renomear arquivo/pasta

### 2. **Ferramentas de Análise** (Úteis)
- `count_lines` - Contar linhas de código no projeto
- `get_imports` - Listar imports/dependencies de um arquivo
- `find_definition` - Encontrar onde função/classe é definida
- `find_usages` - Encontrar onde função/classe é usada

### 3. **Ferramentas Git** (Muito úteis)
- `git_status` - Status do repositório
- `git_diff` - Ver mudanças não commitadas
- `git_log` - Histórico de commits
- `git_blame` - Ver quem modificou linhas

### 4. **Ferramentas de Terminal** (Já parcialmente disponível)
- `run_command` - Executar comando shell
- `install_package` - npm/pip install

**Minha recomendação de prioridade:**

1. **Primeiro:** Ferramentas de modificação (write_file, replace_in_file) - Para a IA poder realmente ajudar a editar código
2. **Segundo:** Ferramentas Git básicas (status, diff) - Para entender contexto de mudanças
3. **Terceiro:** Ferramentas de análise avançada - Para refatoração inteligente
